## 提瓦特观测台 Android v1.1.0

完全移植电脑端 v1.1.0 全部功能。

**布局调整**
- 移除侧边栏和标签页导航
- 改为 PC 版单页滚动布局，所有内容连续展示
- 新增顶部粘性 Header（搜索框 + 主题切换 + 玩家信息）

**主题系统**
- 浅色/深色主题一键切换
- 全部组件使用动态主题颜色
- 星座星空背景跟随主题变化
- CSS 变量 + React Context 双重主题系统

**功能增强**
- 首页内置 UID 搜索表单
- 查询后 Header 右侧显示玩家头像与昵称
- 加载动画（旋转环效果）
- Compass 指南针图标

**安装要求**
Android 7.0+ (API 24+)
